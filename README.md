# NoorBloom — Terms & Privacy

This repository contains static HTML versions of the NoorBloom Terms & Conditions and Privacy Policy for deployment.

Files:

- `index.html` — landing page linking to the legal pages
- `terms.html` — Terms & Conditions (do not modify content unless instructed)
- `privacy.html` — Privacy Policy (do not modify content unless instructed)

Deployment (Netlify):

1. Create a new GitHub repository and push these files.
2. In Netlify, choose "New site from Git" and connect your GitHub repository.
3. Set the build command empty and the publish directory to the repository root (`/`).

If you want, provide the GitHub repo URL and I will add the remote and push the initial commit from this machine.
